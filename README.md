# DocumentLightMarkWipeTool
***
### 使用方法
#### 安装numpy
```
pip install numpy
```
#### 安装PIL
```
pip install pillow
```
#### cd到项目文件夹下执行python DocumentLightMarkWipeTool-python2.py或DocumentLightMarkWipeTool-python3.py
#### 根据提示输入导入和导出的路径，回车执行即可
#### 注意:仅对浅色的黑白/彩色水印有效，如WPS水印，课程水印等
